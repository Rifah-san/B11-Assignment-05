Question-1: What are the different ways to select an element in the DOM?


Five different ways to select an element. 1.getElementById() 2.getElementsByClassName() 3.getElementsByTagName 4.querySelector() 5.querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?


innerHTML Gets the HTML content inside an element. innerText Gets only the visible text inside an element. textContent Gets all text content inside an element, including hidden elements.

Question-3: What is event delegation in the DOM?


Event delegation is a technique a single event listener is used through parent element to handle events for multiple child elements.

Question-4: What is event bubbling in the DOM?


Event bubbling is a propagation mechanism in the DOM where an event first triggers on the target element and then bubbles up to its ancestors

Question-5: How do you create, add, and remove elements using JavaScript?


Create an Element: document.createElement("div"); Add an Element: document.body.appendChild(newDiv); Remove an Element:document.getElementById("Element").remove();
